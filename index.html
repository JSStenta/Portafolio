ScrapingMercado-Back

Descripción:Desarrollo de una API REST en Deno que permite obtener precios de productos de diferentes supermercados mediante web scraping. Facilita la búsqueda de precios actualizados para ayudar a los usuarios a comparar productos entre varias cadenas. Tecnologías: TypeScript, Deno, Oak, Puppeteer. Logro destacado: Implementación de una estructura eficiente y escalable para manejo de datos de supermercados en tiempo real. Repositorio

Descripción: Desarrollo de una interfaz web utilizando React para mostrar precios de productos extraídos con la API ScrapingMercado-Back. Tecnologías: React, CSS, Tailwind. Logro destacado: Integración efectiva de frontend con API backend para visualización de datos en tiempo real. Repositorio
